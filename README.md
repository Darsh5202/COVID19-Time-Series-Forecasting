# COVID19-Data-Analysis-and-Forecasting

Time Series forecasting project for analyzing and predicting COVID-19 cases using Python. This project demonstrates complete data preprocessing, exploratory data analysis (EDA), visualization, and forecasting using multiple statistical and deep learning models.

# 🦠 COVID-19 Data Analysis and Forecasting

## 📌 Project Overview

The COVID-19 pandemic generated massive amounts of time-series data that can be analyzed to understand disease progression and forecast future case trends. This project builds multiple forecasting models to predict COVID-19 confirmed cases using historical data.

The project follows a complete Data Science workflow including data collection, preprocessing, exploratory data analysis, feature preparation, visualization, model training, forecasting, and comparison of different time-series models.

---

## 🎯 Objectives

- Analyze COVID-19 confirmed case trends
- Clean and preprocess time-series data
- Perform Exploratory Data Analysis (EDA)
- Visualize pandemic growth patterns
- Build forecasting models
- Compare statistical and deep learning approaches
- Select the best forecasting model

---

## 📂 Dataset Information

The project uses publicly available COVID-19 time-series datasets containing:

- Confirmed Cases
- Death Cases
- Recovered Cases
- Country/Region
- Province/State
- Latitude
- Longitude
- Daily Date-wise Case Counts
